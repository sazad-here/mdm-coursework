# Linear Programming

**Chapter 12** — Optimization with Excel Solver (Albright 8e).

## How to follow this assignment

| Problem | Page | Dataset in `data/` | What to do |
|---------|------|--------------------|------------|
| 2 | 564 | `Production Mix 1.xls` | Set decision variables, objective, constraints → Solver |
| 11 | 572 | `Production Mix 2.xls` | Product-mix LP; record optimal mix and sensitivity |
| 19 | 580 | `Production Planning Problem19.xls` | Multi-period production planning with Solver |

### Solver checklist

1. Enable **File → Options → Add-ins → Solver**.
2. Identify decision cells, objective cell, and constraints.
3. Run Solver → keep optimal solution.
4. Optionally request **Sensitivity Report**.
