# Regression Fundamentals

**Chapter 9** — Estimating relationships with regression (Albright 8e).

## How to follow this assignment

1. Open a dataset from `data/` in Excel.
2. Build a **scatterplot** (X = predictor, Y = response).
3. Add a **trendline**, show the **equation** and **R²**.
4. Run **Data Analysis → Regression** for simple or multiple regression.
5. Interpret slope, R², and whether the relationship looks linear.
